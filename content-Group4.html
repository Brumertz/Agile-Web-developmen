GROUP OF DEFINITIONS 2

Programming control structures include sequence for linear execution of instructions, selection for conditional branching, and iteration for repeating code blocks. HTML is the standard language for creating and structuring web pages with tags, while CSS is used to style and layout these pages, separating content from presentation. Programming languages like JavaScript add interactivity to web pages, Python is versatile for development and analysis, Java enables cross-platform applications, C# is used for Windows applications and games in Unity, and PHP is for server-side scripting in dynamic web pages.

The three major programming control structures.

  • Sequence: Linear execution of statements.
  • Selection: Conditional branching (e.g., if-else statements).
  • Iteration: Repeated execution of code blocks (e.g., loops).

HTML is the standard language for creating web pages. It uses tags to structure content (e.g., div, h1).

Other markup languages include XML and Markdown, used for different types of data representation.

CSS is used to style and layout web pages. It controls the look and feel of HTML elements, including colours, fonts, spacing, and positioning.

CSS allows for separation of content (HTML) from presentation (CSS), enabling flexible and responsive design.

Programming languages have specific syntax rules that define how code must be written. Uses include:

  • JavaScript: Adding interactivity to web pages.
  • Python: General-purpose programming, popular for web development, data analysis, and automation.
  • Java: Building cross-platform applications, widely used in enterprise environments.
  • C#: Developing Windows applications and games using Unity.
  • PHP: Server-side scripting for dynamic web pages.
